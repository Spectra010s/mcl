# My Campus Library - Project Roadmap

This document outlines the high-priority future features for the platform.

## High Priority (Next Steps)

- [ ] **Advanced Analytics Dashboard**
  - Activity tracking for popular resources.
  - Download and bookmark trends for admins.
  - Course-level engagement metrics.

## Key Features (Ongoing)

- [ ] Advanced search filters (Academic Level, Department)
- [ ] Resource recommendation engine
- [ ] Automated tagging for uploaded materials

## Recently Completed

- [x] **In-browser File Preview (PDF/Image Viewer)**
  - Users should be able to view documents without downloading.
  - Integration with PDF.js or similar for seamless viewing.
- [x] **CBT (Computer-Based Testing) System**
  - Full admin management for tests and questions.
  - Student testing engine with instant scoring.
- [x] **SEO Optimizations**
  - Dynamic `sitemap.xml` generation.
  - Optimized `robots.txt` configuration.
- [x] **Admin Moderation Workflow**
  - Resource approval/rejection system.
  - Email notifications via Resend.
- [x] **User Authentication**
  - Supabase Auth integration.
  - OAuth (Google/GitHub) support.
