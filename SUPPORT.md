## Common Issues

### Login Issues

- Verify your email address
- Check spam folder for verification email
- Try password reset
- Clear browser cookies and cache

### Upload Problems

- Check file size (max 500MB)
- Verify file format (PDF, DOC, etc.)
- Ensure you're logged in
- Check browser console for errors

### Search Not Working

- Try simpler search terms
- Check spelling and course codes
- Clear filters
- Refresh page

## Reporting Security Issues

Found a security vulnerability? Please email [spectra010s@gmail.com](mailto:spectra010s@gmail.com) instead of using public issues.

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if applicable)

## Troubleshooting

### Performance Issues

- Clear browser cache
- Update to latest browser version
- Check internet connection
- Disable extensions

### Data Not Syncing

- Refresh page
- Check network connection
- Sign out and sign in
- Clear browser cache

### Download Issues

- Try different browser
- Check available disk space
- Verify internet connection
- Try different resource

## FAQ

**Q: How do I delete my account?**
A: Go to Settings > Account > Delete Account. This will remove all your data.

**Q: Can I change my username?**
A: Go to Settings > Profile > Change Username This will change yur cureent username to the filled one.

**Q: How long are files stored?**
A: Resources are stored indefinitely unless removed by admin or requested for removal.

**Q: Can I request resource removal?**
A: Yes, report via the help page or contact support.
